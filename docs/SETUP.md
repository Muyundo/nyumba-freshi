# Setup

Windows (PowerShell):

1. Create a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

3. Run the package:

```powershell
python -m nyumba_freshi
```

Notes:
- Add runtime dependencies to `requirements.txt` before installing.
